## 🎯 如何使用本库

### 快速查找

1. **按领域查找** - 从上面的目录中选择相关领域
2. **打开对应目录** - 查看具体的场景示例
3. **复制 Prompt** - 根据你的需求调整并使用
4. **迭代优化** - 根据结果调整参数和约束

### 学习提示

每个示例都包含：
- **场景描述** - 实际应用场景
- **Prompt 模板** - 完整的提示词
- **核心要素** - 关键的输入参数
- **优化建议** - 如何调整以获得更好结果
- **真实示例** - 输入和输出的对比

### 最佳实践

1. **先理解场景** - 读懂为什么这样设计 Prompt
2. **自定义参数** - 根据你的数据调整关键参数
3. **逐步迭代** - 先用简单版本测试，再复杂化
4. **记录改进** - 保存好用的变体版本

## 💡 Prompt 设计模式

本库示例展示了这些常用的 Prompt 设计模式：

- **少样本学习** - 通过示例让 AI 学习模式
- **链式思考** - 分步骤的复杂问题解决
- **结构化输出** - 指定 JSON/表格等输出格式
- **角色扮演** - 给 AI 赋予特定的身份
- **约束条件** - 明确的限制和要求
- **迭代改进** - 逐步优化的策略

## 🚀 获取最大价值

### 第1步：选择场景
找到与你的实际问题最接近的示例

### 第2步：理解原理
阅读"核心要素"部分，理解为什么这样设计

### 第3步：自定义
根据你的具体数据和需求修改参数

### 第4步：测试
用你的数据测试，观察输出质量

### 第5步：优化
基于测试结果调整 Prompt

### 第6步：保存
把好用的版本保存下来，建立个人库

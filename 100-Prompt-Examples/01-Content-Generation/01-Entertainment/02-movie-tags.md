# 🏷️ 电影标签生成

> 为电影自动生成分类标签和词语描述的 Prompt 工程实践

---

## 📌 应用场景

为电影/电视剧自动生成分类标签，用于：
- 影片内容分类和标签系统
- 用户兴趣推荐和筛选
- 搜索和浏览优化
- 内容管理系统（CMS）

---

## 🎯 核心要素

| 要素 | 说明 | 例子 |
|------|------|------|
| **角色** | 电影分类和标签专家 | 内容管理员、推荐算法工程师 |
| **背景** | 电影详细信息 | 片名、类型、情节、演员 |
| **目标** | 生成精准的分类标签 | 多维度标签体系 |
| **约束** | 标签数量、类型、格式 | 5-15个标签，多种维度 |
| **示例** | 标签体系说明 | 类型、情感、主题、人群 |

---

## ✅ 完整 Prompt 模板

### 基础版（简单标签）

```
你是一位电影分类专家。

请根据以下电影信息，生成5-10个标签，用于影片分类和用户搜索。

【电影信息】
- 片名: {电影名称}
- 类型: {电影类型}
- 情节: {简短情节描述}
- 主演: {主要演员}
- 上映年份: {年份}

【标签要求】
1. 数量：5-10个标签
2. 准确性：准确反映影片内容和特点
3. 多维度：包括类型、主题、情感、人物关系等
4. 易搜索：使用用户常见的搜索词

【输出格式】
标签列表：[tag1] [tag2] [tag3] ...

【标签维度说明】
- 类型标签：电影的类型分类
- 主题标签：故事的主要表达主题
- 情感标签：观看时的情感体验
- 人物标签：主角身份或故事角色
- 场景标签：故事发生的背景或设定
```

### 进阶版（多维度标签体系）

```
你是一位资深电影内容管理专家，擅长多维度的影片标签分类。

请根据以下电影信息，从多个维度生成精准的标签体系。

【电影信息】
- 片名: {电影名称}
- 导演: {导演名字}
- 类型: {电影类型}
- 情节梗概: {详细情节描述}
- 主演: {主要演员及角色}
- 上映年份: {年份}
- 豆瓣评分: {评分}（可选）
- 目标受众: {观影人群}

【标签维度体系】

1. **类型维度**（3-4个标签）
   - 一级分类：剧情、喜剧、动作、恐怖等
   - 二级分类：具体细分

2. **主题维度**（2-3个标签）
   - 影片要表达的核心主题
   - 社会议题或人生议题

3. **情感维度**（2-3个标签）
   - 观看时的情感体验
   - 如：温暖、感动、紧张、悬疑等

4. **角色维度**（1-2个标签）
   - 主要人物身份或关系
   - 如：青春爱情、家庭伦理、职场等

5. **背景维度**（1-2个标签）
   - 故事发生的时代或地点
   - 如：民国、古装、现代、校园等

6. **视觉/风格维度**（1个标签，可选）
   - 视觉特效、美学风格
   - 如：视觉震撼、文艺清新等

【输出格式】
**类型标签:** [tag1] [tag2] [tag3]
**主题标签:** [tag1] [tag2]
**情感标签:** [tag1] [tag2] [tag3]
**角色/关系标签:** [tag1] [tag2]
**背景/时代标签:** [tag1] [tag2]
**风格标签:** [tag1]

【生成原则】
- 避免过度标签化
- 确保每个标签都有实际作用
- 优先选择用户常见搜索词
- 突出影片最独特的特征
```

## 🔧 优化建议

### ✅ 做的事情

1. **准确反映核心内容**
   ```
   ✅ 很好：一部剧情片标签包括 [剧情] [家庭] [感动]
   ❌ 不好：标签有 [科幻] [动作] 但电影完全没有这些元素
   ```

2. **使用用户常见搜索词**
   ```
   ✅ 很好：[青春爱情] [校园] [甜蜜]
   ❌ 不好：[迪亚兹的思春期情感表达] （过于冗长和专业）
   ```

3. **突出独特卖点**
   ```
   ✅ 很好：《你的名字。》标签包括 [灵魂互换] [跨越时空]
   ❌ 不好：只有通用标签 [爱情] [日本] [学生]
   ```

4. **多维度覆盖**
   ```
   ✅ 很好：包括类型、主题、情感、背景等多个维度
   ❌ 不好：只关注一个维度（比如只有类型标签）
   ```

### ❌ 避免的事情

1. **过度标签化** - 标签太多反而造成困扰
2. **不准确的分类** - 标签与内容不符
3. **过于专业** - 使用普通用户不会搜索的词
4. **重复标签** - 标签之间含义重复
5. **忽视用户视角** - 不考虑用户搜索习惯

## 📊 标签维度对比

| 维度 | 说明 | 例子 | 用途 |
|-----|------|------|------|
| **类型** | 影片的分类类型 | 剧情、喜剧、动作 | 基础分类和筛选 |
| **主题** | 表达的核心议题 | 爱情、家庭、梦想 | 价值观和兴趣匹配 |
| **情感** | 观看的情感体验 | 感动、搞笑、紧张 | 心情和期待匹配 |
| **角色** | 主角或关系类型 | 青春爱情、职场 | 代入感和代表性 |
| **背景** | 时代和地点设定 | 现代都市、民国、校园 | 兴趣和知识获取 |
| **风格** | 视觉和艺术风格 | 视觉震撼、文艺清新 | 审美偏好匹配 |

## 🚀 快速上手

1. **确定标签维度** - 根据你的需求选择要生成的维度
2. **复制"进阶版"Prompt** - 如果需要完整标签体系
3. **填入电影信息** - 片名、类型、情节等
4. **生成标签** - 得到多维度的标签列表
5. **审核和调整** - 移除不准确的标签，添加遗漏的关键词
6. **上线使用** - 用于分类、推荐和搜索

## 💡 进阶应用

### 应用1：影片推荐系统
```
任务：为推荐系统生成标签，用于用户兴趣匹配

Prompt调整：
- 强调用户会搜索的标签

- 标签要能准确描述影片的核心吸引点

```

```
Prompt调整：
- 标签要简洁易用
- 遵循统一的标签体系
- 支持多语言标签
- 考虑SEO优化
```

```
任务：为浏览页面生成标签，帮助用户快速了解

Prompt调整：
- 突出最吸引用户的特点
- 标签要视觉清晰
- 强调与用户相关的信息
```

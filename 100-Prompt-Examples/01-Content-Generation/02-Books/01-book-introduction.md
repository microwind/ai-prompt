# 📖 书籍简介生成

> 为图书生成吸引人的内容简介的 Prompt 工程实践

---

## 📌 应用场景

为图书生成高质量的内容简介，用于：
- 电子书平台（Kindle、微信阅读等）
- 图书销售平台（亚马逊、豆瓣读书等）
- 出版社宣传文案
- 阅读社区推荐

---

## 🎯 核心要素

| 要素 | 说明 | 例子 |
|------|------|------|
| **角色** | 资深书评人、出版编辑 | 内容策划、文案撰写 |
| **背景** | 书籍基本信息 | 书名、作者、类型、内容 |
| **目标** | 生成引人入胜的简介 | 激发读者购买和阅读欲望 |
| **约束** | 字数、准确性、吸引力 | 150-300字，突出价值点 |
| **示例** | 简介样本 | 主题、情节、收获 |

---

## ✅ 完整 Prompt 模板

### 基础版（简洁简介）

```
你是一位资深书评人和图书编辑。

请根据以下书籍信息，生成一份吸引人的内容简介，用于电子书平台或销售网站。

【书籍信息】
- 书名: {书名}
- 作者: {作者名}
- 出版社: {出版社}
- 出版年份: {年份}
- 类型: {书籍类型：小说/非小说}
- 内容概述: {书籍内容描述}

【简介要求】
1. 字数：150-200字
2. 内容：吸引人的开场 + 书籍核心价值 + 适读人群
3. 语气：专业但不生硬，引发共鸣
4. 亮点：突出作者的独特视角或内容的创新点

【输出格式】
【{书名}】简介

{简介正文}

【适读人群】
- {人群1}
- {人群2}
```

### 进阶版（包含核心亮点分析）

```
你是一位资深出版策划人和文学编辑，擅长为不同类型的书籍创作吸引人的简介。

请根据以下书籍信息，从多个维度生成完整的书籍推荐体系。

【书籍信息】
- 书名: {书名}
- 作者: {作者名}
- 出版社: {出版社}
- 出版年份: {年份}
- ISBN: {ISBN号}（可选）
- 书籍类型: {类型：文学/商业/自助/教育等}
- 字数: {字数}
- 核心主题: {主题}
- 内容概述: {详细内容描述}
- 作者简介: {作者背景}

【简介正文】（200-300字）
请写一份能在书店、线上平台吸引读者的简介，包括：
1. 有趣的开篇吸引注意力
2. 清晰表达书籍的核心价值
3. 强调内容的创新性或独特角度
4. 描绘读者将获得什么

【核心亮点分析】（3-5个）
请列出这本书最突出的特点，每个30-50字：
1. **亮点1**：{描述}
2. **亮点2**：{描述}
3. **亮点3**：{描述}
...

【适读人群分析】
根据书籍内容，列出不同的目标读者：
- **主要读者**：{描述及阅读理由}
- **次要读者**：{描述及阅读理由}
- **潜在读者**：{描述及阅读理由}

【内容亮点引导】
从以下几个维度突出看点：
- 如果你喜欢{相似书籍}，你会爱上这本书因为...
- 如果你关心{话题}，这本书提供了...
- 如果你想要{学习目标}，这本书通过...

【推荐语示范】
请为这本书撰写3-5条推荐语（20-40字每条）：
1. 【简短推荐】{推荐语}
2. 【深度推荐】{推荐语}
3. 【感性推荐】{推荐语}
```

## 💡 真实示例

### 示例1：文学小说

**输入信息：**
```
书名：《活着》
作者：余华
出版社：南海出版社
出版年份：1992
类型：文学小说
内容：讲述一个苦难人生的故事，主人公经历了人生的多次失去。
```

**期望输出：**
```
【《活着》】简介

在时代变迁中，徐福贵从富家少爷沦为贫苦农民，在苦难的人生中，他目睹和经历了无数的失去——失去亲人、失去朋友、失去健康、失去希望。然而，正是这样一个被命运一次次践踏的生命，却用顽强的活着来诠释对生命最深的尊重。这部作品以极简的笔触，却触及了人生最沉重的议题：在绝望中，我们为什么还要活着？

【核心亮点】
1. **生命哲学的深刻思考**：用具体的人生故事探讨生命意义
2. **余华标志性的冷峻笔触**：简洁有力的语言蕴含深层力量
3. **对苦难的尊重和诠释**：不煽情，不廉价，深刻而沉静

【适读人群】
- 对人生意义感到迷茫的读者
- 喜欢深度文学作品的爱好者
- 想要理解中国当代文学的读者
```

### 示例2：商业类书籍

**输入信息：**
```
书名：《从零到一》
作者：彼得·蒂尔
类型：商业/创业
内容：讨论如何从零开始创造真正创新的产品和企业
```

**期望输出：**
```
【《从零到一》】简介

大多数人想的是如何从1到N，即在已有的基础上进行改进和扩展。但真正的创新，是从0到1——凭空创造出一个全新的价值。PayPal联合创始人彼得·蒂尔以其丰富的创业经验和独特的思维框架，揭示了成功创业的本质。从定义初创企业的特征、到识别未来最有潜力的科技方向，这本书挑战了硅谷流传的传统智慧，提供了关于如何在看似不可能的地方找到价值创造的思考。

【核心亮点】
1. **反直觉的思维方式**：颠覆常见的商业认知
2. **系统化的创新框架**：可应用的方法论
3. **来自真实创业者的经验**：PayPal的成功案例证明

【适读人群】
- 有创业想法的企业家和创业者
- 想要突破性创新的企业管理者
- 对硅谷和科技感兴趣的读者

【推荐语示范】
1. "每个想创业的人都应该读这本书" ——亚历克斯·卡普
2. "改变了我对创新的理解，视角独特、观点深刻"
3. "如果你只能读一本关于创业的书，就读这一本"
```

### 示例3：自助励志书

**输入信息：**
```
书名：《习惯的力量》
作者：查尔斯·杜希格
类型：自助类
内容：科学解析习惯的形成机制，以及如何改变不良习惯
```

**期望输出：**
```
【《习惯的力量》】简介

你的很多行为都是习惯——你每天怎么上班、怎么与伴侣互动、怎么思考——而这些习惯往往在你没意识到的时候控制了你的人生。普利策奖得主查尔斯·杜希格用神经科学和心理学的最新研究成果，揭示了习惯的神经学基础：习惯循环由暗示、常规、奖励三个部分组成。通过理解这个循环，你可以有意识地改变习惯，从而改变人生。书中包含大量真实案例，从个人成长到组织变革，展示了习惯改变的力量。

【核心亮点】
1. **科学解析习惯机制**：神经学基础的通俗讲解
2. **实用的改变方法**：真实案例教你如何改变习惯
3. **跨域应用**：从个人到企业，从员工到领导

【适读人群】
- 想要改变某个坏习惯的人
- 对自我提升感兴趣的读者
- 管理者和组织领导者
- 心理学爱好者

【推荐语示范】
1. "这本书改变了我的人生——通过改变我的习惯"
2. "用科学的方式解释习惯，提供可行的改变之道"
3. "比鸡汤文有用，比理论书更有趣"
```

## 🔧 优化建议

### ✅ 做的事情

1. **从读者的需求出发**
   ```
   ✅ 很好："如果你对创新感到困惑，这本书提供了清晰的思考框架"
   ❌ 不好："这本书有400页，讨论了许多关于创新的话题"
   ```

2. **强调书籍的独特价值**
   ```
   ✅ 很好："与其他创业书不同，本书不讲成功故事，而是讲成功的本质"
   ❌ 不好："这是一本关于创业的书"
   ```

3. **多角度呈现价值**
   ```
   ✅ 很好：核心亮点 + 适读人群 + 推荐语多样化
   ❌ 不好：单一的简介描述
   ```

4. **准确描述内容深度**
   ```
   ✅ 很好："书中既有理论框架，也有实践案例"
   ❌ 不好：模糊的"很深刻"或"很有用"
   ```

### ❌ 避免的事情

1. **完全剧透内容** - 保留阅读的惊喜
2. **过度承诺** - 不夸大书籍的作用
3. **行业术语堆砌** - 普通读者难以理解
4. **忽视目标读者** - 不考虑不同人群的需求

## 📊 不同书籍类型的简介重点

| 类型 | 重点 | 语气 | 长度 |
|-----|------|------|------|
| **文学小说** | 情感、主题、文学价值 | 诗意、引发共鸣 | 150-250字 |
| **商业书** | 实用价值、框架、应用 | 专业但易懂 | 150-200字 |
| **自助励志** | 改变潜力、方法论、案例 | 鼓励但真诚 | 150-200字 |
| **专业技术** | 深度、新观点、实用性 | 学术但可读 | 100-150字 |
| **历史传记** | 人物、历史背景、意义 | 叙事、引人入胜 | 150-250字 |

## 🚀 快速上手

1. **准备书籍信息** - 书名、作者、类型、内容等
2. **选择Prompt版本** - 基础版或进阶版
3. **填入书籍细节** - 按照Prompt完成填空
4. **生成初稿** - 获得简介、亮点、推荐语
5. **审核修改** - 确保准确且有吸引力
6. **多平台适配** - 根据平台调整长度
7. **上线使用** - 用于销售或推荐

### 应用1：电子书平台简介

```
- 强调电子阅读的便利性

```

### 应用2：社交媒体推荐
```
任务：为书籍推荐社群生成推荐文

Prompt调整：
- 简洁有力（50-100字）
- 使用社区常见的表达
- 包含#标签和推荐理由
```

### 应用3：出版社营销文案
```
任务：为新书发布生成营销文案

Prompt调整：
- 强调新书的亮点
- 结合作者的品牌价值
- 创造购买紧迫感（如：限量首发）
```

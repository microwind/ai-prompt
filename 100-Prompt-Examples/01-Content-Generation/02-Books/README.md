# 📖 书籍内容生成

> 图书简介、推荐语、作者传记等书籍内容的 Prompt 工程实践

---

## 📚 本目录包含

- **书籍简介生成** - 为书籍生成吸引人的内容简介
- **图书推荐语** - 创意十足的书籍推荐文案
- **作者简历生成** - 作者个人信息卡片
- **章节概述** - 书籍各章内容总结

---

## 🎯 场景应用

这些 Prompt 适用于：
- 📱 电子书平台（Kindle、微信阅读等）
- 🛒 图书销售平台（亚马逊、豆瓣读书等）
- 📰 出版社宣传文案
- 📚 阅读社区推荐
- 🏪 线下书店推荐展示

---

## 💡 快速开始

1. 选择需要的文件（如 `01-book-introduction.md`）
2. 打开文件查看完整的 Prompt 模板
3. 根据你的书籍信息填充参数
4. 将 Prompt 复制到 AI 工具中使用
5. 根据输出调整和优化

---

**📝 注**: 所有示例都包含 Prompt 模板、实际案例和优化建议。

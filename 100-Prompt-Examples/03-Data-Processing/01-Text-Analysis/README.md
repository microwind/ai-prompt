# 📊 文本分析

> 对文本进行深度分析、提取信息和识别特征的 Prompt 工程实践

---

## 📚 本目录包含

- **情感分析** - 分析评论和反馈的情感倾向
- **关键词提取** - 从文本中识别核心主题词
- **文本分类** - 自动对文本进行分类
- **实体识别** - 识别文本中的人名、地点、机构等
- **文本摘要** - 提炼长文本的核心内容

---

## 🎯 场景应用

这些 Prompt 适用于：
- 💬 用户评论和反馈分析
- 📝 文本分类和标签
- 📰 新闻内容分析
- 🏷️ 文本关键词提取
- 📊 用户舆情监测

---

## 💡 快速开始

1. 选择需要的分析类型（如情感分析）
2. 打开对应的文件查看完整 Prompt
3. 准备待分析的文本
4. 将 Prompt 填充后复制到 AI 工具
5. 根据结果采取行动

---

**📝 注**: 每个示例都包含基础版和进阶版两个 Prompt 模板，适应不同复杂度的需求。

# 📺 电视剧简介生成

> 为电视剧生成故事梗概和剧集简介的 Prompt 工程实践

---

## 📌 应用场景

为电视剧生成吸引人的故事简介和剧集概述，用于：
- 流媒体平台剧集介绍
- 电视剧海报和宣传
- 用户了解剧情和进展
- 剧集导航和追剧指南

---

## 🎯 核心要素

| 要素 | 说明 | 例子 |
|------|------|------|
| **角色** | 电视剧编审和宣传文案撰写 | 内容编辑、剧评人 |
| **背景** | 电视剧基本信息 | 剧名、类型、集数、情节纲要 |
| **目标** | 生成引人入胜的简介 | 吸引用户开始或继续观看 |
| **约束** | 字数、准确性、无剧透 | 100-200字，不剧透重要转折 |
| **示例** | 简介示范 | 情节、情感、故事线 |

---

## ✅ 完整 Prompt 模板

### 基础版（单集简介）

```
你是一位电视剧编审和宣传文案撰写者。

请根据以下电视剧信息，生成一份吸引人的剧集简介。

【电视剧信息】
- 剧名: {电视剧名称}
- 类型: {剧集类型}
- 第{X}集（或季）
- 本集/季主要情节: {情节描述}
- 关键角色: {主要人物}

【简介要求】
1. 字数：100-150字
2. 内容：介绍本集/季的主要看点和情感线
3. 吸引力：激发观众继续观看的欲望
4. 避免剧透：不透露重要转折或结局

【输出格式】
【{剧名} 第X集简介】
{简介内容}

【看点提示】
- {看点1}
- {看点2}
- {看点3}
```

### 进阶版（整部剧简介 + 剧情发展线）

```
你是一位资深电视剧编审和内容策划专家。

请根据以下电视剧信息，生成一份完整的剧集简介体系。

【电视剧信息】
- 剧名: {电视剧名称}
- 导演/编剧: {主创信息}
- 类型: {剧集类型}
- 总集数: {集数}
- 主演: {主要演员及角色}
- 播出年份: {年份}
- 故事背景: {时代、地点、设定}
- 核心故事线: {主线情节概述}

【整体简介】（150-200字）
请写一份吸引人的整体简介，包括：
1. 故事的核心设定和背景
2. 主要角色的身份和关系
3. 中心矛盾或核心冲突
4. 整部剧的情感基调

【主要剧情线索】
请列出主要的3-5条故事线索，每条200字左右：
1. **线索1**（角色：{角色名}）
   - 开始情节
   - 发展高潮
   - 预期方向

2. **线索2**（角色：{角色名}）
   - 开始情节
   - 发展高潮
   - 预期方向

...以此类推

【剧情进展地图】
请按集数/季数标注主要转折点：
- 第1-5集：{主要事件}
- 第6-10集：{主要事件}
- 第11-15集：{主要事件}
...

【人物关系图简述】
用简洁的语言描述主要角色之间的关系：
- {角色A} 与 {角色B}：{关系描述}
- {角色C} 与 {角色D}：{关系描述}
...

【观看建议】
根据观众不同需求提供观看路线：
- 追求爱情线的观众：重点看{XX剧情}
- 喜欢悬疑的观众：重点看{XX剧情}
- 追求笑点的观众：重点看{XX剧情}
```

## 💡 真实示例

### 示例1：单集简介

**输入信息：**
```
剧名：《爱的迫降》
第一集
情节：北韩特种兵李准赫在一次空降训练中被吹偏，意外降落在南韩财阀千金姜哈娜的私人高尔夫球场，两人因此相识。
主演：玄彬、孙艺珍
```

**期望输出：**
```
【爱的迫降 第一集简介】

命运的转折就在瞬间发生。一次突如其来的坠落，让北韩特种兵李准赫从广阔的天空坠落到南韩千金姜哈娜的生活中。两个本该毫无交集的人，因为一个意外相识。他是纪律严明的军人，她是自由任性的富家女。当东西方的差异、身份的鸿沟、祖国的分裂在他们之间展开时，爱情究竟还能坚持多久？

【看点提示】
- 跨越三八线的爱情开端
- 玄彬和孙艺珍的绝妙化学反应
- 身份背景带来的冲突与碰撞
```

### 示例2：整部剧简介

**输入信息：**
```
剧名：《鬼怪》
导演：李应福
类型：奇幻爱情
总集数：16集
主演：孔刘、金高银、王世子
背景：现代韩国，融合传说和现实
核心故事：孤独的鬼怪和投生的女神的爱情故事
```

**期望输出：**
```
【整体简介】

他是在人间游荡了900年的孤独鬼怪，等待那个能拔出剑、为他结束生命的人。她是在天堂投生的灵魂，却遗忘了前世的记忆。当这两个被命运诅咒的人相遇时，爱情成为了打破诅咒的唯一可能。《鬼怪》以奇幻的设定融合现实的温暖，讲述了关于救赎、陪伴和爱的故事。这是一部在笑中有泪、在泪中有爱的韩剧经典。

【主要剧情线索】

1. **鬼怪的诅咒与救赎**（角色：金信）
   - 开始情节：900年孤独的鬼怪在等待救赎
   - 发展高潮：与女神的相识让他看到希望
   - 预期方向：爱情能否打破千年诅咒

2. **女神的轮回与记忆**（角色：智硕）
   - 开始情节：投生为人类的她遗忘了一切
   - 发展高潮：逐渐发现自己真实身份
   - 预期方向：记忆恢复后会如何选择

3. **死神的孤独与救赎**（角色：亚冬）
   - 开始情节：冷漠的死神邂逅温暖的青年
   - 发展高潮：被人类温度所触动
   - 预期方向：跨越阴阳的感情能否成立

【人物关系图简述】
- 金信（鬼怪） 与 智硕（女神）：前世宿命的重逢，现世的爱恋
- 亚冬（死神） 与 旺京（少年）：冷漠与温暖的碰撞
- 旺京（少年） 与 德铉（女医）：现世中的青葱爱情
```

## 🔧 优化建议

### ✅ 做的事情

1. **设置悬念而非透露情节**
   ```
   ❌ 不好："她最终发现自己是转世的女神，但鬼怪为了拯救她而消失了"
   ✅ 很好："被命运诅咒的两个人，爱情能否打破千年的枷锁？"
   ```

2. **强调情感线而非情节线**
   ```
   ❌ 不好："第一集讲的是鬼怪坠落和女神出现"
   ✅ 很好："命运的转折在瞬间发生，两个孤独的灵魂因为意外而相识"
   ```

3. **不同受众的观看路线**
   ```
   ✅ 很好：
   - 喜欢爱情的观众→看女神和鬼怪的感情线
   - 喜欢幽默的观众→看死神和旺京的互动
   - 喜欢奇幻的观众→看诅咒、轮回等设定
   ```

4. **准确描述剧集特色**
   ```
   ✅ 很好："奇幻设定、爱情线、温暖人心、致郁情节"
   ❌ 不好：太笼统的描述
   ```

### ❌ 避免的事情

1. **过度剧透关键转折** - 保留悬念
2. **过于简洁** - 不足以让观众了解看点
3. **情节细节堆砌** - 简介应该有吸引力而非信息列举
4. **忽视观众需求** - 不考虑不同观众的兴趣

---

## 📊 简介长度建议

| 类型 | 场景 | 字数 | 内容重点 |
|-----|------|------|---------|
| **单集简介** | 平台剧集页 | 80-120字 | 本集看点，不剧透 |
| **季简介** | 新季开播 | 150-200字 | 季度主题和转折 |
| **整部简介** | 初接触用户 | 200-300字 | 整体设定、吸引点 |
| **推荐理由** | 社交分享 | 50-80字 | 最核心的吸引点 |

---

## 🚀 快速上手

1. **选择简介类型** - 单集？整季？还是整部？
2. **准备基本信息** - 剧名、类型、人物、情节等
3. **复制对应Prompt** - 基础版或进阶版
4. **填入信息** - 根据Prompt完成填空
5. **生成简介** - 获得初稿
6. **审核修改** - 确保准确性和吸引力
7. **上线使用** - 用于平台或宣传

## 💡 进阶应用

### 应用1：追剧进度提示
```
任务：为追剧者生成每周更新提示

Prompt调整：
- 强调本周关键转折
- 提醒观众可能的剧情发展
- 无关键剧透，保留悬念
```

### 应用2：新手入坑指南
```
任务：为想开始看的人生成看剧指南

Prompt调整：
- 强调为什么要看这部剧
- 说明需要的观看时间投入
- 描述可能的情感体验
```

### 应用3：剧集回顾总结
```
任务：为已看完的人生成故事总结

Prompt调整：
- 可以包含关键剧透
- 强调关键转折点
- 反思人物成长和故事主题
```

---

**💡 提示**: 一份好的电视剧简介，应该既能吸引新观众入坑，也能让已看过的人回味故事的美妙。

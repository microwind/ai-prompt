# 🎬 电影推荐语生成

> 为电影生成海报推荐文案的 Prompt 工程实践

---

## 📌 应用场景

为电影/电视剧海报生成引人入胜的一句话推荐文案，用于：
- 电影平台推荐卡片
- 海报宣传文案
- 精选推荐栏目
- 用户发现页面

---

## 🎯 核心要素

| 要素 | 说明 | 例子 |
|------|------|------|
| **角色** | 电影宣传文案撰写者 | 资深影评人、营销专家 |
| **背景** | 电影基本信息 | 片名、类型、情节概述 |
| **目标** | 生成吸引人的推荐文案 | 一句话，控制字数，突出卖点 |
| **约束** | 字数、语气、关键词 | 15-30字、情感化、突出亮点 |
| **示例** | 好的文案例子 | "绝望中的光芒，他们用爱拯救彼此" |

---

## ✅ 完整 Prompt 模板

### 基础版（简单场景）

```
你是一位资深电影宣传文案撰写者。

请根据以下电影信息，生成一句话推荐文案（15-30字），用于电影海报。

【电影信息】
- 片名: {电影名称}
- 类型: {电影类型}
- 情节: {简短情节描述}
- 主演: {主要演员}

【要求】
1. 字数：15-30字
2. 语气：吸引人、有情感冲击
3. 突出卖点：情节亮点、演员阵容或视觉效果
4. 避免：过于直白、剧透重要情节

【输出格式】
推荐文案：[你的推荐文案]
文案亮点：[说明这个文案的1-2个特点]
```

### 进阶版（考虑目标受众）

```
你是一位资深电影营销专家，了解不同观众群体的审美偏好。

请根据以下信息，为不同目标受众生成个性化的推荐文案。

【电影信息】
- 片名: {电影名称}
- 导演: {导演名字}
- 类型: {电影类型}
- 情节梗概: {详细情节描述}
- 核心主题: {影片想表达的主题}
- 视觉/音乐亮点: {技术层面的优势}

【目标受众分析】
- 主要受众：{目标1}（年龄、兴趣）
- 次要受众：{目标2}（年龄、兴趣）
- 潜在受众：{目标3}（年龄、兴趣）

【要求】
1. 为每个受众生成一句推荐文案（15-30字）
2. 突出对该受众最有吸引力的卖点
3. 使用该受众喜欢的表达方式和情感诉求
4. 确保文案新颖、避免重复

【输出格式】
**针对{受众1}:**
推荐文案：[文案内容]
卖点分析：[为什么这个文案对此受众有吸引力]

**针对{受众2}:**
推荐文案：[文案内容]
卖点分析：[为什么这个文案对此受众有吸引力]

**针对{受众3}:**
推荐文案：[文案内容]
卖点分析：[为什么这个文案对此受众有吸引力]
```

## 💡 真实示例

### 示例1：文艺电影

**输入信息：**
```
片名：《又见奈良》
类型：文艺剧情片
情节：两个曾经相爱的人，在中年时在古都奈良重逢，面对过往和遗憾，最终学会珍惜当下。
主演：林青霞、田中哲司
```

**期望输出：**
```
推荐文案：时光轮回，爱在重逢时绽放，林青霞用眼神诠释中年女性的故事。
文案亮点：
- 强调时间与爱情的循环主题
- 突出林青霞的表演力量
- 营造诗意、遗憾又温暖的情感氛围
```

### 示例2：动作大片

**输入信息：**
```
片名：《速度与激情10》
类型：动作冒险片
情节：全球团队对抗终极反派，为了拯救世界进行惊险任务。
主演：范迪塞尔、杰森·莫玛
```

**期望输出：**
```
推荐文案：全球通缉，终极对决，为了世界和家族，他们无所畏惧。
文案亮点：
- 突出高风险、高刺激的动作设定
- 强调"家族"和"正义"的主题
- 营造紧张、燃爆的观影期待
```

### 示例3：喜剧电影

**输入信息：**
```
片名：《一只狗的使命2》
类型：温馨喜剧
情节：一只狗通过多次转世，与主人完成心灵救赎，理解什么是真正的陪伴。
主演：乔什·盖德（配音）
```

**期望输出：**
```
推荐文案：一生有你，它用九条命诠释什么叫"陪伴"，催泪又治愈。
文案亮点：
- 突出"陪伴"和"救赎"的温暖主题
- 暗示重生/多次转世的独特设定
- 同时营造温馨和感动的双重情感
```

## 🔧 优化建议

### ✅ 做的事情

1. **突出独特卖点** - 找到这部电影与众不同的地方
   - 特殊的视角或背景
   - 演员的突破性表演
   - 独特的视觉或音乐风格

2. **激发情感** - 而非仅描述情节
   ```
   ❌ 不好："这是一部关于爱情的电影"
   ✅ 很好："在最绝望的时刻，他用爱给了她重生的理由"
   ```

3. **避免剧透** - 保持神秘感
   ```
   ❌ 不好："他们最终发现彼此的身份竟然是……"
   ✅ 很好："真相让所有人都措手不及"
   ```

4. **考虑受众** - 不同人群有不同诉求
   - 青少年：冒险、视觉特效、明星
   - 家长：教育意义、家庭主题
   - 文艺青年：深度、艺术价值

### ❌ 避免的事情

1. **过度承诺** - 文案与电影实际效果差异大
2. **模板化表达** - 使用过度使用的套路
3. **忽视目标受众** - 用老年人的语言卖给年轻人
4. **冗长** - 超过30字失去冲击力

### 场景1：电影平台推荐卡片
```

- 片名：《你的名字。》
- 目标受众：年轻学生、动漫爱好者

请生成一句推荐文案，要求：

- 融入动漫特有的浪漫情调
- 激发年轻观众的观看欲望

```

```
- 片名：《怦然心动》
- 目标：吸引情侣档观众
- 应用：电影院海报宣传

Prompt修改：
请生成一句推荐文案用于电影院海报，要求：
- 体现甜蜜爱情的主题
- 适合贴在电影院海报区域
- 强调两个主演的化学反应
- 20-30字，温馨治愈的语调
```

## 📊 对比：好文案 vs 不好的文案

| 方面 | 不好的文案 | 好的文案 | 原因 |
|-----|---------|--------|------|
| **情感** | "这是一部爱情电影" | "他们用爱改写了彼此的人生" | 后者激发情感，更有吸引力 |
| **焦点** | "演员表演很好，剧情不错" | "林青霞用沉默诠释中年女性的秘密" | 后者更具体、更有视觉感 |
| **承诺** | "最好看的电影之一" | "如果你看过《城市猎人》，你会爱上这部" | 后者更可信，针对性更强 |
| **字数** | 45字左右 | 20-28字 | 更容易记住和传播 |

## 🚀 快速上手

1. **复制上面的"进阶版"Prompt**
2. **填入你的电影信息**（片名、类型、情节）
3. **描述目标受众**（年龄、兴趣、观影习惯）
4. **调整字数和语调**（根据平台需求）
5. **生成结果** → 获得个性化推荐文案
6. **选择最佳方案** → 上线使用

---

**💡 提示**: 最有效的推荐文案往往是那些能够唤起特定情感、而非简单描述情节的文案。

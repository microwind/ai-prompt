## 💡 快速开始

1. 选择需要生成的代码类型
2. 打开对应的文件查看完整 Prompt
3. 描述你的需求（功能、字段、样式等）
4. 将 Prompt 复制到 AI 工具
5. 得到代码框架或完整实现
